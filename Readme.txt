
Running main.py will generate the outputs folder.

The files under the outputs folder are described as follows:
continuous_scaler.pkl and model.pkl are model files;
evaluate.txt contains the model evaluation results;
confusion_matrix.png is the confusion matrix image;
roc.png is the ROC curve image;
voice.png, voice_log_mel.png, and voice_mfcc.png are the waveform, Mel spectrogram, and MFCC images of the first audio, respectively.
